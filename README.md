## Theme 

```css
- Fond principal de l'application : #FFFFFF (Blanc pur pour un maximum de clarté).

- Sidebar (Barre latérale) : #4A4A4A (Anthracite pour donner de la structure et un côté premium).

- Texte et icônes de la Sidebar : #CBCBCB (Gris clair pour un contraste doux).

- Lien actif ou Hover dans la Sidebar : #FFFFFF avec un fond léger ou texte passant en #FFFFE3.

- Header (Barre supérieure) : #FFFFFF avec une fine bordure basse #CBCBCB pour séparer proprement.

- Contenu des cartes (Cards) : #FFFFE3 (Le blanc ivoire sert ici de fond pour les zones de contenu importantes comme l'agenda ou les notes, ce qui les fait ressortir subtilement).

- Boutons principaux : #6D8196 (Le bleu-gris attire l'action) avec texte #FFFFFF.

- Textes principaux du contenu : #000000 ou #4A4A4A.
```
